## 1. 产品概述

仓库货位热力 3D 查看器——面向仓储管理人员的可视化工具，将货架布局与拣货频率数据结合，以 3D 热力图形式直观呈现货位热度，辅助拣货路径优化与异常排查。

- 核心问题：传统 WMS 系统缺少空间维度的拣货频率可视化，难以快速定位高频/低频货位及布局异常
- 目标用户：仓库主管、物流分析师、仓储系统管理员

## 2. 核心功能

### 2.1 功能模块

1. **3D 热力场景页**：可旋转/缩放的货架 3D 渲染，货位按拣货频率着色，支持日期与区域筛选
2. **筛选与配置面板**：日期范围选择器、区域多选、颜色阈值滑块、相机书签管理
3. **异常检测面板**：未知货位标注列表、坐标冲突行级错误列表、异常清单导出

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 3D 热力场景页 | 货架渲染区 | 根据布局数据渲染 3D 货架，货位盒子按热力值着色（冷→热：蓝→绿→黄→红） |
| 3D 热力场景页 | 交互控制 | OrbitControls 旋转/缩放/平移，点击货位显示详情 Tooltip |
| 3D 热力场景页 | 筛选栏 | 日期范围选择、区域下拉多选、实时过滤热力数据 |
| 3D 热力场景页 | 阈值配置 | 低/中/高三档阈值滑块，实时更新热力着色 |
| 3D 热力场景页 | 相机书签 | 保存/恢复相机视角，书签列表可删除 |
| 3D 热力场景页 | 数据导入 | 上传布局 JSON 和拣货记录 JSON，内置样例数据 |
| 异常检测面板 | 未知货位列表 | 拣货记录引用的不存在于布局中的货位 ID，红色标记 |
| 异常检测面板 | 坐标冲突列表 | 布局中同行同坐标的货位，行级报错，有效货位仍渲染 |
| 异常检测面板 | 导出功能 | 导出异常清单为 JSON 文件 |

## 3. 核心流程

用户导入布局和拣货数据 → 系统校验数据（检测未知货位、坐标冲突） → 有效货位渲染为 3D 货架 → 拣货频率映射为热力颜色 → 用户通过筛选/阈值交互探索 → 保存视角书签和配置 → 重启后恢复全部状态

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色工业风（#0f1419 背景 + #1a2332 卡片）+ 亮青色强调（#00d4ff）
- **热力色谱**：蓝（#3b82f6）→ 绿（#22c55e）→ 黄（#eab308）→ 红（#ef4444）
- **字体**：JetBrains Mono（数据/代码区域）+ 系统默认 UI 字体
- **布局**：左侧窄面板（筛选+异常） + 右侧宽幅 3D 画布
- **按钮风格**：圆角 8px，半透明背景 + 边框发光效果
- **图标**：lucide-react 图标库

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 3D 热力场景页 | 3D 画布 | Three.js 透视相机，OrbitControls，网格地面，环境光+方向光 |
| 3D 热力场景页 | 左侧面板 | 日期选择器、区域复选框、阈值滑块、相机书签卡片 |
| 3D 热力场景页 | 顶部工具栏 | 导入按钮、样例数据按钮、导出异常按钮、全屏切换 |
| 3D 热力场景页 | 异常面板 | 红色/黄色标签的错误行、点击定位到3D对应位置 |
| 3D 热力场景页 | Tooltip | 悬浮卡片显示货位ID、区域、坐标、拣货次数 |

### 4.3 响应式

- 桌面优先（1920×1080 基准）
- 最小宽度 1024px，3D 画布自适应剩余空间
- 面板可折叠

### 4.4 3D 场景指引

- **环境**：深色背景，微弱网格地面，淡蓝色雾效
- **光照**：环境光 0.4 强度 + 两个方向光（主光偏暖，补光偏冷）
- **相机**：透视相机，FOV 50，初始俯视 45° 角度
- **货架**：半透明框架线 + 实心货位方块，方块颜色按热力值映射
- **交互**：OrbitControls 旋转/缩放，hover 高亮货位，点击弹出 Tooltip
- **后处理**：轻微辉光（Bloom）效果增强热力视觉冲击

## 5. 数据格式

### 5.1 布局数据（layout.json）

```json
{
  "locations": [
    { "id": "A-01-01", "zone": "A", "row": 1, "col": 1, "layer": 1, "x": 0, "y": 0, "z": 0 },
    { "id": "A-01-02", "zone": "A", "row": 1, "col": 2, "layer": 1, "x": 1.2, "y": 0, "z": 0 }
  ]
}
```

### 5.2 拣货记录（picks.json）

```json
{
  "records": [
    { "locationId": "A-01-01", "timestamp": "2024-03-15T08:30:00Z", "quantity": 5 },
    { "locationId": "UNKNOWN-01", "timestamp": "2024-03-15T09:00:00Z", "quantity": 3 }
  ]
}
```
